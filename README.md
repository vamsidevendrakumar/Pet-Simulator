## ✨ Features

- 🎮 **Interactive Gameplay** - Engage with your virtual pet in real-time
- 🎨 **Beautiful UI** - Modern and responsive user interface
- ⚡ **Fast Performance** - Built with Vite for lightning-fast development and builds
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices
- 🎯 **Easy to Extend** - Clean Vue 3 component architecture for easy customization

## 🚀 Quick Start

### Prerequisites
- Node.js >= 14.0
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/vamsidevendrakumar/Pet-Simulator.git
cd Pet-Simulator

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
